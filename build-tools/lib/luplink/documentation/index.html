ngx-luplink documentation

Developping a library : you might need to use npm-link in order to avoid the process of building the library & restarting the application each time you want to modify or debug the library. See : https://medium.com/dailyjs/how-to-use-npm-link-7375b6219557 npm-link isn't commited with git so if you want to use it you will need to set it up by yourself. You can easily set do the link with :

bash link.sh

In order to use the library on local NPM, you first have to build it: ng build --project=ngx-luplink

You can then pack it by going to dist/ngx-luplink and executing npm pack. This creates a package that can be install using npm install path/to/ngx-luplink.tgz


#Activate npm-link
cd project/ngx-luplink
npm link
cd ../luplink-app
npm link ngx-luplink
